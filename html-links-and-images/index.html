This link is created using the "a" tag.

The HTML attribute "href" inside the beginning "a" tag will let you place a link inside for the anchor to redirect to. (This one redirects to YouTube.)

This link will open in a new tab. This link has the "target" HTML attribute set to "_blank", meaning it will open YouTube in a new window or tab.

This link will open in this tab. This link has the "target" HTML attribute set to "_self", meaning it will open YouTube in the current tab.

This link will open in a new tab. This link has the "rel" HTML attribute set to "noopener", meaning the opened link will not gain access to the webpage from which it was opened.

This link will open in a new tab. This link has the "rel" HTML attribute set to "noreferrer", meaning the opened tab will not know the webpage it was opened from. (includes functionality of "noopener")

Noopener prevents phishing attacks, where the opened link will change the original website to a scam website in case you want to go back.

However, "_blank" in most browsers takes on the attributes of "noopener" and "noreferrer" anyway.

This is an absolute link, pointing to https://youtube.com. It will typically be in the format, protocol://domain/path.
